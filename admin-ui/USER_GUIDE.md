# คู่มือการใช้งาน Admin UI

## 1. คำอธิบายระบบ

โปรเจกต์นี้เป็น Control Panel สำหรับการจัดการและควบคุมระบบการปรับปรุงฐานข้อมูล MySQL โดยมีฟีเจอร์หลักๆ ดังนี้:
- การจัดการการเชื่อมต่อฐานข้อมูล
- การแสดงคำแนะนำการปรับปรุง
- การจัดการการดำเนินการ (Executions)
- การควบคุม Kill Switch

## 2. การติดตั้งระบบ

### 2.1 ต้องการเครื่องมือ
- Node.js (เวอร์ชัน 18 หรือสูงกว่า)
- npm (เวอร์ชัน 8 หรือสูงกว่า)

### 2.2 ขั้นตอนการติดตั้ง
1. ติดตั้ง Node.js และ npm จากเว็บไซต์อย่างเป็นทางการ
2. เปิด terminal แล้วเข้าไปที่โฟลเดอร์ `admin-ui`
3. รันคำสั่ง:
   ```
   npm install
   ```

## 3. การใช้งานระบบ

### 3.1 การรันระบบ
- รันในโหมดพัฒนา:
  ```
  npm run dev
  ```
- คอมไพล์สำหรับใช้งานจริง:
  ```
  npm run build
  ```

### 3.2 โครงสร้างหน้าเว็บ
- **Dashboard**: หน้าแรกแสดงภาพรวมของระบบ
- **Connections**: การจัดการการเชื่อมต่อฐานข้อมูล
- **Recommendations**: คำแนะนำการปรับปรุง
- **Executions**: การจัดการการดำเนินการ
- **Kill Switch**: การควบคุมระบบ

## 4. ปัญหาที่พบและวิธีแก้ไข

### 4.1 ปัญหาการติดตั้งแพ็กเกจ
หากพบข้อผิดพลาดเกี่ยวกับการติดตั้งแพ็กเกจ `lucide-react` หรือ `react-icons`:

1. ตรวจสอบว่า Node.js และ npm ติดตั้งแล้วหรือยัง
2. รันคำสั่ง:
   ```
   npm install lucide-react
   ```
   หรือ
   ```
   npm install react-icons
   ```

### 4.2 ปัญหาการคอมไพล์
หากพบข้อผิดพลาดในการคอมไพล์:
1. ตรวจสอบว่าไฟล์ `package.json` มีการระบุแพ็กเกจที่จำเป็นแล้วหรือยัง
2. รันคำสั่ง `npm install` เพื่อติดตั้งแพ็กเกจที่ขาดหายไป

## 5. คำแนะนำการใช้งาน

- ระบบจะทำงานในโหมดอ่านอย่างเดียวโดยค่าเริ่มต้น
- การดำเนินการใดๆ ต้องมีการยืนยันจากผู้ใช้
- ทุกการคลิกจะถูกบันทึกใน audit log
- ระบบจะแสดงคำเตือนความเสี่ยงก่อนการดำเนินการใดๆ
